# Rust Foundation

## Purpose

This directory contains the foundation outputs for the Rust implementation of Anvil.

The goal of this phase is to define the initial Rust structure clearly enough that implementation can start without collapsing into a monolith.

## Scope

This phase covers:

- repository layout
- module boundaries
- app bootstrap
- config model
- logging and error model
- initial implementation sequence

It does not yet implement the full product.
